Note: 
HotCorners always run on startup automatically, to disable it delete              
"HotCorners.exe" form startup folder of your windows or simply type
"Startup" in your windows search bar.

---> HotCorners always starts in system tray and can be 
     controlled from there with the help of GUI

---> Restore option in GUI closes the app to make effects to take place
     , it needs to be restarted manually.
